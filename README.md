# Task 7 — Basic Sales Summary (SQLite + Python)

## Overview
This project uses Python and SQLite inside a Jupyter Notebook to generate a simple sales summary. It shows how to:
- Create a small SQLite database (sales_data.db)
- Insert sample sales data
- Run SQL queries from Python
- Load results into pandas
- Display a summary table and a bar chart

## Files Included
- task7dataxlabs.ipynb — Main Jupyter Notebook
- sales_data.db — SQLite database created in the notebook
- sales_chart.png — Bar chart saved from the notebook

## How to Run
1. Open Jupyter Notebook
2. Load task7dataxlabs.ipynb
3. Run all cells from top to bottom

The notebook will:
- Create the database
- Insert sample data
- Execute the SQL summary
- Print the results
- Generate and save sales_chart.png

## Expected Output (DataFrame)
   product  total_qty  revenue  
0  Gadget          3     75.0  
1  Widget          5     50.0  
2  Doodad          4     20.0  

## Expected Output (Chart)
A bar chart saved as sales_chart.png showing revenue by product.

## Purpose
This task demonstrates:
- Basic SQLite usage
- Running SQL from Python
- Simple data analysis with pandas
- Creating basic charts

Useful for beginner data analysis and GitHub portfolio practice.
